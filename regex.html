Regular Expressions in JavaScript

Regular expressions are a powerful tool for working with text in JavaScript. They allow you to search for and manipulate text based on patterns. Here's a list of some of the most commonly used regular expressions in JavaScript:

Expression Description
/pattern/ A pattern to match a specific sequence of characters
/^pattern/ Matches the pattern only at the beginning of a string
/pattern$/ Matches the pattern only at the end of a string
/[abc]/ Matches any one of the characters inside the brackets
/[^abc]/ Matches any character not inside the brackets
/.*/ Matches any character zero or more times
/[a-z]/ Matches any lowercase letter
/[A-Z]/ Matches any uppercase letter
/[0-9]/ Matches any digit
/\s/ Matches any whitespace character
/\S/ Matches any non-whitespace character
/\d/ Matches any digit
/\D/ Matches any non-digit character
/\w/ Matches any word character (letters, digits, and underscores)
/\W/ Matches any non-word character
/x+/ Matches the character 'x' one or more times
/x*/ Matches the character 'x' zero or more times
/x?/ Matches the character 'x' zero or one time
/x{5}/ Matches the character 'x' exactly 5 times
/x{2,5}/ Matches the character 'x' between 2 and 5 times
/x{2,}/ Matches the character 'x' 2 or more times
/(x)/ Captures the character 'x' and stores it in a group
/(?:x)/ Matches the character 'x' without storing it in a group
/(x)(y)/ Captures the characters 'x' and 'y' and stores them in separate groups
/x(?=y)/ Matches the character 'x' only if it is followed by the character 'y'
/x(?!y)/ Matches the character 'x' only if it is not followed by the character 'y'
